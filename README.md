# Time Sessions Landing Page

A modern, responsive landing page for the Time Sessions time tracking app.

## Features

- **Responsive Design**: Works beautifully on desktop, tablet, and mobile devices
- **Modern UI**: Built with Tailwind CSS for a clean, professional look
- **Smooth Animations**: Engaging hover effects and smooth scrolling
- **SEO Optimized**: Proper meta tags and semantic HTML
- **Fast Loading**: Minimal dependencies, uses CDN for Tailwind CSS

## Structure

- `index.html` - Main landing page
- `logo.svg` - App logo (activity-flow-icon.svg)
- `favicon.png` - App icon for browser tab

## Sections

1. **Hero Section**: Eye-catching introduction with CTA buttons
2. **Features Section**: Highlights 6 key features with icons
3. **How It Works**: 3-step process explanation
4. **Screenshots Section**: Visual showcase of the app
5. **Download CTA**: Prominent download section with app store buttons
6. **Footer**: Links and social media

## Customization

### Colors
The landing page uses Time Sessions' brand colors:
- Primary: `#3B82F6` (Blue)
- Accent: `#4C1D95` (Purple)

You can customize these in the Tailwind config section of `index.html`.

### Content
Update the following:
- App store download links (currently placeholder `#`)
- Social media links in the footer
- Add actual app screenshots if desired
- Update the favicon.png with your app icon

## Deployment

This is a static HTML page that can be deployed to:
- **Netlify**: Drag and drop the folder
- **Vercel**: Connect to your repository
- **GitHub Pages**: Push to a gh-pages branch
- **Any static hosting service**

## Local Development

Simply open `index.html` in your browser. No build process required!

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript (for smooth scrolling)
- Google Fonts (Inter)
